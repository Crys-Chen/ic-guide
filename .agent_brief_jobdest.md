# 任务简报：科研方向「代表性机构（就业去向）+ 顶会顶刊」改写

你是本仓库（MkDocs ECE 自学指南）的内容编辑 agent。任务是把 `docs/科研方向/<方向>.md` 里现有的 `## 代表性机构` 小表格，**展开重写**成「就业去向」式的分级列表，并把顶会顶刊**单独拆成一个 section**。

## 总目标

告诉读者「学了这个方向之后能去哪」——分**企业**和**科研院所**两类，国内/国外都要，**主要覆盖中国和美国**（其他重要的海外机构如欧洲/日韩/台湾/港校酌情纳入）。每条单独换行，带**官网超链接**；上市公司再附一个**实时股价链接**。

## 最终版式（严格遵守）

把原来的 `## 代表性机构` 整段（含表格）替换为下面这个结构，并在其后**紧接**新增一个 `## 顶会顶刊` section（放在原 `## 知识路径` 之前）：

```markdown
## 代表性机构

> 这个方向毕业后能去的代表性企业与科研院所（国内外）。上市公司附实时股价链接，便于了解产业景气度。

### 企业

**国内**

- [公司名](官网URL) · [实时股价](股价URL)
- [未上市公司名](官网URL)（未上市）

### 科研院所

**国外**
```

### 版式硬性要求
1. **企业**和**科研院所**必须分成两个 `###` 子块。
2. 每个子块内再用加粗 `**国内**` / `**国外**` 分组。
3. **每个企业/机构单独一行**（`- ` 列表项），不要挤在一行用顿号堆。
4. 每条**必须有官网超链接**（`[名字](URL)`）。
5. 顶会顶刊**必须从代表性机构里拆出来**，单独成 `## 顶会顶刊`，并把「顶会」和「顶刊/期刊」分开（原表里多数只有顶会，请补充该领域公认顶刊；硬件/电路类常见顶刊如 JSSC、TCAS、TED、TVLSI、TCAD 等，按方向选真实的）。
6. 数量：**企业每个区域（国内/国外）至少 2–3 个**，多则多放（产业大的方向如 AI/处理器/器件可放 5–8 个）。科研院所国内外各列该方向公认的代表（国内 2–4，国外 2–4）。
7. 保留原表已有的、确实正确的条目，在其基础上**补全**（补官网链接、补股价、补科研院所、补国内外缺口）。

## 股价链接规则（按市场分源）

只给**上市主体**加股价链接。代码/交易所**必须真实**，不确定就用 WebSearch 核实，**禁止编造代码**。

- **A股·上交所**（6 开头，含科创板 688）：`https://quote.eastmoney.com/sh<6位代码>.html`
  - 例：中芯国际 688981 → `https://quote.eastmoney.com/sh688981.html`
- **A股·深交所**（0/2/3 开头，含创业板 300）：`https://quote.eastmoney.com/sz<6位代码>.html`
  - 例：韦尔股份 603501 是上交所→sh603501；澜起科技 688008→sh688008；北方华创 002371→sz002371
- **A股·北交所**（8 开头）：`https://quote.eastmoney.com/bj<6位代码>.html`
- **港股**：`https://quote.eastmoney.com/hk/<5位代码>.html`
  - 例：中芯国际港股 0981 → `https://quote.eastmoney.com/hk/00981.html`；腾讯 0700 → `.../hk/00700.html`
- **美股**：`https://finance.yahoo.com/quote/<TICKER>`
  - 例：NVIDIA → `https://finance.yahoo.com/quote/NVDA`；台积电 ADR → `https://finance.yahoo.com/quote/TSM`
- **其他海外**（用 Yahoo Finance 带交易所后缀）：
  - 韩国 KRX：`<代码>.KS`（三星电子 005930.KS → `https://finance.yahoo.com/quote/005930.KS`）
  - 日本东证：`<代码>.T`（东京电子 8035.T）
  - 台湾证交所：`<代码>.TW`（台积电本土 2330.TW）；台积电也可直接用美股 ADR `TSM`
  - 欧洲：优先用其美股 ADR（如 ASML 在纳斯达克 `ASML`、英飞凌 ADR `IFNNY`、意法半导体 `STM`）
- **未上市 / 母公司未上市**：不给股价链接，官网链接后用纯文本注明，如 `（未上市）` 或 `（华为未上市）`、`（OpenAI 未上市）`。
  - 若该业务是某上市母公司的一个部门（如 华为海思属华为=未上市；TSMC 的子业务仍归 TSMC 上市），按实际：能映射到上市主体就给该主体股价，否则标未上市。

## 正确性约束（不可妥协，作者反复强调「不要幻觉乱编」）

1. **官网 URL 必须真实可访问**。用 WebFetch/WebSearch 核实，尤其是国内公司官网（很多是 .com/.cn，别想当然）。验证返回正常（非 404/停靠页）。
2. **股票代码必须真实**，核实交易所与代码再拼 URL。
3. 公司/院所必须**确实与该方向相关**——别把通用大厂硬塞进不相关方向。判断该方向做什么，参考页面开头 `## 这个方向在研究什么` 那几段。
4. 科研院所指**研究机构/国家实验室/高校实验室/公司研究院**（如中科院微电子所、之江实验室、北京智源、MIT CSAIL、Stanford SystemX、imec、AIST、Fraunhofer 等），不是高校招生意义上的「学校」——高校具体课题组已在页面后面的 `## 相关课题组` 里，**这里不要和课题组重复堆教授名单**，列的是机构层面。
