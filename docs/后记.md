# 后记

> 国内绝大部分大学的本科教学，不是濒临崩溃，而是早已崩溃。
> ——《上海交通大学学生生存手册》

> 本科教育至少得展现它应有的诚意，一种分享知识的诚意，一种以人为本的诚意，一种注重学生体验的诚意。它至少不应该是一种恶意，一种拼比知识的恶意，一种胜者为王的恶意，一种让人学无所得的恶意。但这一切能改变吗？我不知道。
> ——《CS自学指南》

第一次知道北大那份《CS自学指南》，是 2023 年打复旦系际辩论赛的时候。当时有一道辩题是"当今中国，加强通识教育还是专业教育更有利于大学生发展"，我们持方是反方，也就是“加强专业教育”。有一条论证走的是"缺啥补啥"：当前内地的本科教育，根本没能给人才提供正常的学科训练和专业培养，连北大那种顶级学府的 A+ 学科都需要学生写一份《CS自学指南》来自救。

2024 年初，我萌生了如法炮制一份《IC自学指南》的念头。这个念头倒逼我多听多想，多整理笔记和资源，断断续续，做了两年半，最终汇聚成了现在这个网站。最初我只想搭一个小小的个人主页，把高年级攒下的一点认知和笔记分享出去，范围局限于复旦。可慢慢地我觉得这样没意思，毕竟在这个专业里挣扎的远不止复旦学子。要是哪天有其他学校的同学从社交媒体慕名点进来，发现满屏资源都是复旦专属，那该有多失落！所以最终它长成了现在的样子。

搭网站这件事对我并不容易，我并非学计算机或软件工程的，Coding 对我来说只是工具，但并不精通。前端、UI、衬线还是非衬线、按钮要不要圆角，每一样都琐碎得磨人，光做一个轮播图就能耗掉我一整天。这也是这个网站耗了我两年半的一大原因。是 Claude Code 把我从这些泥潭里捞了出来。最近一年 AI 的变化，我体感很真切。去年暑假我还在硬着头皮手撸代码，那时 AI Coding 错漏百出，与其给它擦屁股不如我自己来。但到了今年年初，Claude Code 已经可以彻底优化掉码农。就像复旦陈鸣飞在脱口秀说的，感觉自己就像“勾践尝了三年胆，发现夫差让别人消灭了。人轻松了，苦白吃了！”

“晚清刚进宫的太监，一个阉人，准备工作刚做完，发现行业消失了”

磨的过程里，我始终在想一件事，就是"我想成为的人"和"系统想把我塑造成的人"之间，隔着多大的张力。管教育的人嘴上说要培养什么样的人才，可他们落到纸面的制度、培养方案和激励机制，常常南辕北辙。单看培养方案，系统就是要我们面面俱到。可"生也有涯，知也无涯"，要是我老老实实照系统的要求，每一门课都掏出十足的精力去学、去刷高分，那我的大学生活怎么办？

有人说，一所理想的大学，本该是一间自助餐厅。把资源尽可能多地摆出来，我们身在其中，各取所需就好，不必总想着把学生捏成某个固定的形状。可现实是，"国内绝大部分大学的本科教学早已崩溃"这句话写于 2008 年，敢问今夕何年，情况可有好转，诸君冷暖自知。我也在国外游历过一些学校，发现欧美的本科教育其实同样在崩。世道如此，倒也不必苛责系统里某个具体位置上的人。说到底，"本科"和"教育"这两个词，越来越不像是能安心放在一起说的了；如今还在认真琢磨"本科教育"四个字的，恐怕只剩做宣传的那群人。那么，如今的大学，还能给我们留下点什么？

自己读《》，还是去上赵惠俊老师的《宋词导读》呢？

我想，是"传道"。"师者，所以传道授业解惑也"。可如今工科九成的线下课都不如一门好网课，九成九的助教都没有 AI 那么耐心，"授业"和"解惑"早就守不住了。守得住的，只剩"传道"。本科这几年，我还是实实在在遇到过一些扛得起"传道"二字的老师，谢锡麟、郑益薏（交大）、易婷、蒋玉龙、严昌浩、叶凡、茹国平、许灏、王伶俐。他们站在讲台上的时候，嘴角是上扬的。或许讲课这件事，本就不必那么追求效率。就像写字，对今天的很多人来说已经成了一桩纯粹的消遣；上课也一样，学知识的路子有很多，课堂只是其中一条。能坐在这些老师的课上，本身就是一种享受。
