# 大语言模型

LLM 是当前 AI 系统研究的主要工作负载,训练和推理的算力规律直接决定加速器设计,因此从深度生成模型目录独立成组。

## 推荐课程

- **[清华刘知远 OpenBMB:大模型公开课](OpenBMB_LLM.md)** — 中文入门,大模型全景
- **[李沐:深度学习论文精读](limu_paper_reading.md)** — 经典论文逐篇精读,建立读论文的方法论
- **[Stanford CS336: Language Modeling from Scratch](Stanford_CS336.md)** — 从零写出一个 LLM,动手深度最大
- **[CMU 11-711: Advanced NLP](CMU_11-711.md)** — 研究生 NLP 全景课

## 学习路径建议

先有[深度学习](../深度学习/index.md)基础,至少懂 Transformer。中文入门走 OpenBMB,想动手实现选 CS336。

- [处理器架构与编译系统](../../../科研方向/处理器架构与编译系统.md) — Transformer 加速器设计
