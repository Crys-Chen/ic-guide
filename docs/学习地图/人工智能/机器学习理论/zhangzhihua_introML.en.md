# Zhang Zhihua: Introduction to Machine Learning

## Descriptions

- Offered by: Shanghai Jiao Tong University / Peking University
- Prerequisites: Probability, linear algebra
- Programming Languages: None (theoretical)
- Instructor: Prof. Zhang Zhihua
- Difficulty: 🌟🌟🌟🌟
- Estimated Hours: ~35–42 lectures

The sister course to Zhang Zhihua's *Statistical Machine Learning*. Covers **reproducing kernels / dimensionality reduction / EM / matrix completion / discriminant analysis / SVM / boosting**. Emphasizes "machine learning is the love between statistics and computation" — theoretical and algorithmic derivations in equal measure.

A good ramp-up before the statistical ML course.
