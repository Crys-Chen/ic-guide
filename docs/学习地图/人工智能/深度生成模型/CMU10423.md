# CMU 10423: Generative AI

## 课程简介

- 所属大学：卡内基梅隆大学（CMU）
- 先修要求：机器学习基础、深度学习（Transformer 架构）
- 编程语言：Python / PyTorch
- 课程难度：🌟🌟🌟🌟
- 预计学时：60 小时

CMU 面向本科高年级与研究生开设的生成式 AI 课程，覆盖扩散模型、VAE 等生成模型基础，并着重讲授大语言模型（LLM）的对齐技术（RLHF/DPO）。风格偏向宏观综述，每讲列举大量相关工作，适合快速建立生成模型领域全貌认知后，再针对感兴趣子方向深入。课程作业设计有趣，编程量适中，适合作为 MIT 6.S978 或 UCB CS294-158 的补充。
