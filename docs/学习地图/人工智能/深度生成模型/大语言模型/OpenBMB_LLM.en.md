# Tsinghua · Liu Zhiyuan / OpenBMB: Large Language Model Open Course

## Descriptions

- Offered by: Tsinghua NLP / OpenBMB
- Prerequisites: Deep learning, PyTorch, Transformer
- Programming Languages: Python (PyTorch, HuggingFace)
- Instructors: Liu Zhiyuan's team
- Difficulty: 🌟🌟🌟
- Estimated Hours: ~20 hours (137 sections)

The most systematic Chinese-language LLM open course. **Nine chapters** spanning fundamentals, key techniques, and cross-disciplinary applications — from PyTorch / HuggingFace through **Prompt / Delta Tuning / RAG / MoE / Scaling Law**. Carefully edited and subtitled by the OpenBMB team.

The most authoritative Chinese-language LLM course; complements Stanford CS336 — CS336 builds language models from scratch, OpenBMB leans toward applications and engineering.
