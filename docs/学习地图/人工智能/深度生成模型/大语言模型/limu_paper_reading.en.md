# Mu Li: Deep Learning Paper Reading (incl. LLM Series)

## Descriptions

- Offered by: Boson AI / formerly Amazon
- Prerequisites: Deep learning basics
- Programming Languages: None (paper-reading focused)
- Instructor: Mu Li
- Difficulty: 🌟🌟🌟🌟
- Estimated Hours: 30+ episodes

Mu Li's **paragraph-by-paragraph close-reading** of landmark papers — Transformer / BERT / GPT-3 / CLIP / Llama 3.1 and beyond. Widely dubbed "the workhorse of Bilibili's learning section". The go-to Chinese-language resource for graduate students learning *how* to read frontier LLM papers.

This is not a "systematic course" but a "how-to-read-papers" methodology — after finishing it you know how to read NeurIPS / ICML / ICLR papers properly.
