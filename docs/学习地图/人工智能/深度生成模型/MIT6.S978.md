# MIT 6.S978: Deep Generative Models

## 课程简介

- 所属大学：麻省理工学院（MIT）
- 先修要求：深度学习基础（神经网络、反向传播）、概率论
- 编程语言：Python / PyTorch
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：80 小时

何恺明（Kaiming He）于 2024 年新开的生成模型课程，系统覆盖当前主流深度生成模型的基础理论：变分自编码器（VAE）、生成对抗网络（GAN）、归一化流（Normalizing Flow）、分数匹配与去噪扩散模型（DDPM/Score Matching），以及自回归模型。课程注重数学原理推导，作业提供丰富脚手架代码，既加深理论理解，又不至于在工程实现上过度受阻。

与 MIT 6.S184（侧重 Flow/Diffusion 数学视角）互补，是对生成模型全貌有系统学习需求的首选课程。
