# CMU · Tianqi Chen: Machine Learning Compilation (MLC, Chinese Edition)

## Descriptions

- Offered by: CMU / Apache TVM community
- Prerequisites: Deep learning, Python, C++
- Programming Languages: Python (TVM Unity)
- Instructor: Tianqi Chen
- Difficulty: 🌟🌟🌟🌟
- Estimated Hours: 11 lectures / ~30 hours

A Chinese-language systems course taught by Tianqi Chen in Summer 2022. Built around four abstractions — **computational graph, tensor program, operator library, hardware instructions** — to convey TVM Unity and the MLC philosophy. Paired with Jupyter Notebook labs; the most authoritative Chinese-language tutorial on edge AI deployment.

Complements the existing "AI Computing Systems" entry: AICS covers the full software-hardware stack, MLC focuses on the compiler abstraction.
