# Mu Li: Dive into Deep Learning v2 (PyTorch Edition)

## Descriptions

- Offered by: Amazon / UC Berkeley (formerly MXNet author Mu Li)
- Prerequisites: Linear algebra, Python
- Programming Languages: Python (PyTorch)
- Instructor: Mu Li
- Difficulty: 🌟🌟🌟
- Estimated Hours: ~70+ lectures

Mu Li's "Learning AI with Mu Li" live-stream series. Each chapter pairs model principles with hands-on PyTorch implementation — from **MLP / CNN / RNN / Attention** through CV and NLP applications. The tightest-coupling Chinese-language "models + systems + engineering" practical course, designed as a companion to the *Dive into Deep Learning* textbook.
