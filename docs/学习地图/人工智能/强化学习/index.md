# 强化学习

强化学习（Reinforcement Learning）研究智能体如何在与环境的交互中通过奖励信号学习最优策略。在芯片方向，RL 的应用场景正在快速扩展：**ML for EDA**（布局布线优化、参数调优）、**AI 芯片架构搜索**（NAS / 硬件参数搜索）、**具身智能**（机器人控制策略）。

## 推荐课程

- **[西湖赵世钰：强化学习的数学原理](xhu_zhao_RL.md)** — 中文，贝尔曼方程到 Actor-Critic 严格推导，配开源教材 ★
- **[David Silver：UCL RL](silver_RL.md)** — 英文，AlphaGo 设计者，重直觉，10 讲入门经典 ★

## 学习路径建议

**快速入门**：Silver 10 讲（约 15 小时）建立直觉框架

**做研究用**：赵世钰 54 讲 + 配套教材，读完能直接看 PPO / SAC 原论文

两门合在一起大约 6–8 周。先修要求：[概率统计](../../数学/数学进阶/概率统计/)（马尔可夫链的概率论基础）。

## 对应的科研方向

- [EDA 与设计自动化](../../../科研方向/EDA与设计自动化.md) — RL 用于布局（Google Chip Placement with RL, Nature 2021）
- [具身智能](../../../科研方向/具身智能.md) — 机器人控制策略的主要算法范式
- [AI 算法与系统](../../../科研方向/AI算法与系统.md) — RL 是大模型对齐（RLHF）的核心算法
