# 强化学习

智能体在交互中学策略的一支,具身智能的控制核心,也用于 EDA 布局布线等决策问题。

## 推荐课程

- **[西湖赵世钰:强化学习的数学原理](Westlake_zhao_RL.md)** — 中文,从数学原理推起,底子打得牢
- **[David Silver: UCL Reinforcement Learning](silver_RL.md)** — 经典英文课,框架建得全

深度强化学习课 [UCB CS285](../深度学习/UCB_CS285.md) 在深度学习目录。
