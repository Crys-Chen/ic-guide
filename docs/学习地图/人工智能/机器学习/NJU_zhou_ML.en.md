# Nanjing University · Zhou Zhihua: Introduction to Machine Learning ("Watermelon Book" Companion)

## Descriptions

- Offered by: Nanjing University
- Prerequisites: Linear algebra, probability, Python
- Programming Languages: Python
- Instructor: Prof. Zhou Zhihua
- Difficulty: 🌟🌟🌟
- Estimated Hours: ~52 lectures

The official MOOC personally taught by Zhou Zhihua (academician), companion to his textbook *Machine Learning* (the Chinese "Watermelon Book"). Covers **model evaluation, linear models, decision trees, SVMs, neural nets, Bayesian methods, ensemble learning, and clustering** — the most authoritative version, taught by the author himself.
