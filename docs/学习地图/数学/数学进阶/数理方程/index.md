# 〔征集中〕数理方程（ODE/PDE）

> 这个槽位虚位以待。如果你熟悉该领域，欢迎通过[参与建设](../../../../参与建设.md)推荐课程，一起完善这份学习地图。

## 为什么需要它

服务方向：[半导体器件与先进工艺](../../../../科研方向/半导体器件与先进工艺.md)、[MEMS与微纳传感器](../../../../科研方向/MEMS与微纳传感器.md)。

器件 TCAD 仿真解的是漂移-扩散方程，电磁仿真解的是麦克斯韦方程组，MEMS 多物理场仿真解的是弹性与热传导方程。从数学分析到数值分析之间，缺的就是这一环。
