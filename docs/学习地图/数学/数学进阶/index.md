# 数学进阶

## 学习路径建议

**按科研方向反推**:
- 做 ML/AI:概率统计 + 矩阵分析 + 凸优化
- 做模拟/射频:复变函数 + 信号处理基础
- 做 EDA:离散数学 + 凸优化 + 数值分析
- 做硬件安全/密码:抽象代数 + 密码学 + 信息论
- 做量子计算:线性代数(深入)+ 概率 + 量子力学

不必把所有进阶数学都学一遍。
