# MIT 18.701/18.702: Algebra I/II (Artin)

## 课程简介

- 所属大学：MIT
- 先修要求：MIT 18.700（线性代数，proof-based）
- 编程语言：无
- 授课教师：Michael Artin
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：两学期 ~80 学时

由代数几何大师 Michael Artin（Emil Artin 之子）亲授的 MIT 数学系本科 honors 抽代序列。最大特点是**从几何直觉切入**——用矩阵群、对称性、晶格群讲群论，不像传统抽代直接从置换群开始。配套教材是 Artin 自己写的《Algebra》（Harvard Math 122 用同一本书），公认抽代教材里"最有 taste"的一本。

OCW 只提供 lecture notes 和作业，**无视频**——但 Artin 的讲义本身就足以独立学习。
