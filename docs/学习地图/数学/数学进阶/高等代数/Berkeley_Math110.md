# UC Berkeley Math 110: Linear Algebra

## 课程简介

- 所属大学：UC Berkeley
- 先修要求：Math 53/54（多变量微积分 + 入门线代）
- 编程语言：无
- 授课教师：常由不同教授开（Paulin / Stankova / Borcherds 等）
- 课程难度：🌟🌟🌟🌟
- 预计学时：一学期 ~40 学时

Berkeley 数学系本科核心线代/高代课，proof-based。从向量空间到内积空间、对偶空间、Jordan 标准型、二次型，匹配中国"高等代数（上下册）"的全部内容。讲义在 Berkeley 数学系老师个人主页常公开（Paulin 版讲义口碑最好，可作主参考）。

与 MIT 18.700 互为替代，Math 110 略偏几何与多线性代数，对二次型和正交群讲得更细。
