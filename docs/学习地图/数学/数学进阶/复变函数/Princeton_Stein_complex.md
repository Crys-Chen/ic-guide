# Princeton · Stein《Complex Analysis》（教材）

## 课程简介

- 所属大学：Princeton（Stein-Shakarchi 教材）
- 先修要求：实分析、拓扑入门
- 编程语言：无
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：因人而异（自学教材）

Princeton 四卷分析系列第二卷 *Complex Analysis*。从全纯函数推到 ζ 函数与素数定理，覆盖深度远超本科复变课程，是研究生级别的标准参考书。

适合数学专业本科高年级 / 数学研究生，或对 Stein 系列有兴趣的工科学生。注意这是一本教材而非视频课程，目前没有与之配套的公开课程录像，需要视频讲解的同学建议先看本目录下的西交复变或高数叔系列。
