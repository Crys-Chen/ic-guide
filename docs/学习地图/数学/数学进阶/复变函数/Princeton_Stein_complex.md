# Princeton · Stein 复分析（视频配套）

## 课程简介

- 所属大学：Princeton（基于 Stein-Shakarchi 教材）
- 先修要求：实分析、拓扑入门
- 编程语言：无
- 授课教师：社区/学生录制讲解
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：YouTube 完整 playlist

Princeton 四卷分析系列第二卷 *Complex Analysis* 的 YouTube 视频配套（社区/学生录制讲解版）。从全纯函数推到 ζ 函数与素数定理，是研究生级别的复变课程，覆盖深度远超本科。

适合数学专业本科高年级 / 数学研究生，或对 Stein 系列有兴趣的工科学生。
