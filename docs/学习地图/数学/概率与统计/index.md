# 概率与统计

概率论是机器学习、随机信号、通信和良率分析的共同基础。

## 推荐课程

- **[浙大:概率论与数理统计](ZJU_probability.md)** — 国内教材体系的中文配套课
- **[Harvard Stat 110](Harvard_Stat110.md)** — Blitzstein 主讲,例题驱动,直觉建立得快
- **[MIT 6.041](MIT_6.041.md)** — 概率系统分析,工程视角
- **[MIT 6.262](MIT_6.262.md)** — 研究生随机过程;本站 17 个方向里直接用到它的很少,做通信/排队论建模再看

## 学习路径建议

[入门速成](../入门速成/index.md)的工程数学及概率方法和 CS70 已覆盖工科够用的概率子集。要系统学,浙大或 Stat 110 选一门跟完即可。
