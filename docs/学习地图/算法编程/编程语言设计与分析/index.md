# 编程语言设计与分析

编程语言设计与分析(PL)研究**编程语言的语义、类型系统、程序分析与变换**。它和"编译原理"互补:编译原理偏工程实现(词法、语法、IR、代码生成),PL 偏理论(类型论、形式语义、程序证明)。

对硬件研究者来说,PL 是**做现代编译器(LLVM/MLIR/TVM)、形式验证、硬件描述语言研究**的理论基础。如果你打算做 EDA 中的逻辑综合、形式等价性检查,或做 AI 编译器,PL 知识能让你看懂论文中的数学符号。
