# 软件工程

软件工程教**如何写"工业级"代码**——测试驱动开发、规约设计、模块化、版本控制、code review、维护性。它不是"另一门编程语言",而是**让你的代码能被别人读、能持续迭代、能在大型团队中存活**的方法论。

对 ECE 研究者来说,软件工程在两个场景非常关键:
1. **EDA 工具开发** — Yosys、OpenROAD、Verilator 都是百万行级 C++ 工程,没有软件工程素养无从下手
2. **科研代码可复现性** — 一个写得清楚的 GitHub repo 比 10 篇论文更能让你的工作被引用

- **[UCB CS169](CS169.md)** — Berkeley 软工课,偏 Agile / 敏捷开发
- **[CMU 17-803](17803.md)** — CMU 软件工程研究生课
