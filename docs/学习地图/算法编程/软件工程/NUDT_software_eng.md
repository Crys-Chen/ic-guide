# 国防科技大学：软件工程（国家精品）

## 课程简介

- 所属大学：国防科技大学
- 先修要求：编程入门
- 编程语言：无
- 授课教师：齐治昌、毛新军等
- 课程难度：🌟🌟🌟
- 预计学时：50 学时

国家精品开放课，国防科大软件工程的"老牌标杆"。毛新军教授还是教育部"101 计划"软件工程课程虚拟教研室负责人。系统覆盖软件过程、需求工程、设计、测试、维护。
