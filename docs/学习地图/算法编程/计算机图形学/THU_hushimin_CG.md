# 清华大学 胡事民：计算机图形学基础（国家精品）

## 课程简介

- 所属大学：清华大学
- 先修要求：线性代数、C++/Python
- 编程语言：C++ / Python
- 授课教师：胡事民、刘永进、张松海、徐昆
- 课程难度：🌟🌟🌟
- 预计学时：28 讲 / 32 学时

胡事民教授（中国科学院院士）领衔的国家级精品课，含清华图形学顶尖团队。是 GAMES101 之外最经典的中文图形学体系课，更偏数学基础与传统渲染管线。
