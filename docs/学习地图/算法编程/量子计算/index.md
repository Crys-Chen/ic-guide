# 量子信息与量子计算

量子计算利用量子叠加与纠缠实现经典计算机无法高效完成的计算——Shor 算法破解 RSA 的指数加速、Grover 算法的平方根加速、HHL 算法求解线性方程组。对微电子方向的同学，量子计算的意义是双重的：它是**量子芯片（超导、离子阱、光子）的软件侧**，也是密码学（后量子密码）的对手模型。

学量子计算的先修是线性代数的复数部分，不需要物理系的量子力学——这里的数学框架完全基于线性算子与概率测量。
