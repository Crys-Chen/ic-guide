# Peking University · Guo Wei: Programming and Algorithms (Part III) — Object-Oriented C++

## Descriptions

- Offered by: Peking University
- Prerequisites: C language
- Programming Languages: C++
- Instructor: Prof. Guo Wei
- Difficulty: 🌟🌟🌟
- Estimated Hours: 8 weeks

Prof. Guo Wei's 15-year-running C++ course at PKU. The final assignment is a *World of Warcraft* gameplay simulator (notoriously demanding), paired with his textbook *New Standard C++ Programming*.

Among the most recommended Chinese-language C++ courses for systematic learners following an intro C course.
