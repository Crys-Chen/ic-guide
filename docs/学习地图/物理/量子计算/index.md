# 量子计算

前置是[量子力学](../量子力学/index.md),至少学到会用 Dirac 记号做计算。两门课一中一英,一个建算法框架,一个带代码实操。

## 推荐课程

- **[北大李彤阳:量子计算](PKU_li_quantum.md)** — 量子算法系统课(Grover/Shor/量子复杂度)
- **[Qiskit: Understanding Quantum Information and Computation](qiskit_watrous.md)** — IBM 官方课程,Watrous 主讲,公式推导配代码实操

## 学习路径建议

先李彤阳建框架,再用 Qiskit 把算法跑起来。微电子学生关心的低温控制电路、量子比特读出电路属于科研方向的内容,见下。
