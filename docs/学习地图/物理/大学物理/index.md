# 大学物理

大学物理(力学 + 电磁学 + 光学 + 热力学)是所有理工科本科生的共同物理基础。对 IC 学生来说,大物的**电磁学章节**最直接相关——它是后续学习电路分析、信号传输、射频天线、半导体物理的物理底子;**光学章节**则是进入光电子/硅光方向的入门。

## 推荐课程

- **[大学物理 B(复旦)](PHYS120013-14.md)** — 复旦校内课,知识点对照培养方案
- **[MIT 8.01/8.02: Classical Mechanics & Electromagnetism](MIT8.01-8.02.md)** — Walter Lewin / Peter Dourmashkin 主讲,YouTube + B 站全公开,**国外名校大物的标杆**

## 学习建议

- 国内大物通常一学期讲完力学+电磁+热学+光学,深度不及 MIT 两学期;**有志于做器件、模拟 IC、射频、光电子的同学,把 MIT 8.02(电磁)完整学完远比国内大物刷题更值**
- 数学工具不熟:先看 [数学分析](../../数学/数学基础/数学分析/MITmaths.md)、[线性代数](../../数学/数学基础/线性代数/MITLA.md)
- 光学章节内容浅尝辄止,要做硅光方向的同学请直接看 [光学板块](../光学/index.md)
