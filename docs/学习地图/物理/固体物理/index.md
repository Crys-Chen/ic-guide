# 固体物理

固体物理研究**晶体中电子和声子的行为**——晶格振动、电子能带、布里渊区、输运现象。它在物理知识链上**承上启下**:上承量子力学(把单电子薛定谔方程推广到周期势阱中的多体系统),下启半导体物理(直接给出"能带"这一半导体器件的核心概念)。

IC 学生如果想做器件、工艺、光电子方向,**绕不开固体物理**——你看到的 N 型/P 型掺杂、PN 结、肖特基势垒,本质都是固体物理在半导体材料上的具体应用。
