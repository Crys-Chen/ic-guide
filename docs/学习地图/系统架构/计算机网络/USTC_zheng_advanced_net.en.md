# USTC · Zheng Quan: Advanced Computer Networks

## Descriptions

- Offered by: USTC
- Prerequisites: Introductory computer networks
- Programming Languages: None
- Instructor: Prof. Zheng Quan
- Difficulty: 🌟🌟🌟🌟
- Estimated Hours: ~50 hours

The advanced sequel to Prof. Zheng Quan's undergraduate networking course at USTC, covering **SDN / QUIC / BBR / multimedia networking / mobile wireless networks** — a rare systematic "advanced networking" course in Chinese.

Good follow-up after HIT's introductory networks course for diving into modern networking.
