# GPU 体系结构

GPU 从图形加速器演化为通用并行计算引擎，背后是一套独特的微架构设计哲学：**牺牲控制流灵活性换取吞吐率**——数千个简单核心按 SIMT 方式执行，延迟由深度多线程隐藏。理解 GPU 微架构是研究 AI 系统、近存计算、GPU 互联的前提。
