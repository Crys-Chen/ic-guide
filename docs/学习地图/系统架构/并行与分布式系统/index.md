# 并行与分布式系统

并行与分布式系统研究**如何让多个计算单元协同工作**:GPU 并行编程、分布式一致性、消息传递、容错。这是现代 AI 训练(数千 GPU 集群)、大规模数据处理(Spark/Flink)、区块链(分布式共识)的共同基础。

对硬件研究者来说,并行 + 分布式是**理解 AI 系统、大规模计算的必备背景**。GPU 内核、互联网络、HBM 内存协议——这些硬件细节最终都要为分布式负载服务。
