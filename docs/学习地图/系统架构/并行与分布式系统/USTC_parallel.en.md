# USTC: Parallel Computing (National Quality Course)

## Descriptions

- Offered by: USTC
- Prerequisites: Operating systems, computer architecture
- Programming Languages: C / MPI / OpenMP
- Instructors: USTC parallel computing team
- Difficulty: 🌟🌟🌟
- Estimated Hours: ~40 hours

A national-class quality course that organically integrates **parallel machine architecture, parallel numerical algorithms, and parallel programming** (MPI/OpenMP) — the most classic Chinese-language parallel computing intro.
