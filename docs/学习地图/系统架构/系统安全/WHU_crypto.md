# 武汉大学：密码学（国家精品）

## 课程简介

- 所属大学：武汉大学（国家网络安全学院）
- 先修要求：离散数学、数论基础
- 编程语言：无
- 授课教师：武大网安学院团队
- 课程难度：🌟🌟🌟🌟
- 预计学时：约 40 学时

武大国家网络安全学院的国家精品课，系统讲基本理论与现代密码协议。是国内信息安全/密码学方向最权威的中文公开课，配合 CS161/CS366 等英文系统安全课构成完整体系。
