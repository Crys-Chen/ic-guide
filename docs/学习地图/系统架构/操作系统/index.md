# 操作系统

操作系统研究**软件如何管理硬件资源**:进程调度、内存管理、文件系统、I/O 子系统、虚拟化、中断处理。它是系统类研究的"中间层":上接编译/应用,下连体系结构。

对硬件方向的同学来说,操作系统是**理解软硬件协同**的关键——你设计的硬件特性(缓存替换策略、虚拟内存、加速器接口)最终都需要 OS 适配才能被应用使用。

## 推荐课程

- **[MIT 6.S081](MIT_6.S081.md)** — MIT 经典 OS 课,xv6 教学操作系统(Unix 风格)读+写完整 ★
- **[UCB CS162](UCB_CS162.md)** — Berkeley OS,Pintos 项目
- **[NJU OS](NJU_OS.md)** — 南京大学 蒋炎岩 主讲,**国内中文 OS 课的最高水平** ★;深入讲并发、虚拟化等现代主题
- **[HIT OS](HIT_OS.md)** — 哈工大李治军,Linux 0.11 内核精讲

## 学习路径建议

**入门:MIT 6.S081 或 NJU OS** → 选一门完整做完 Lab/Project,读+写一遍内核代码,对 OS 的认知会有质变。

国内同学强烈推荐 NJU OS——蒋老师讲法现代,Lab 设计精巧,中英文资料结合无障碍。
