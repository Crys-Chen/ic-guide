# 数据库系统

数据库系统研究**如何高效地存储、查询、更新结构化数据**:存储引擎、查询优化、并发控制、事务、列存/行存,以及现代云原生数据库、向量数据库等。

对硬件研究者来说,数据库的**索引数据结构、并发控制、I/O 优化**是计算机系统的精华案例;**做存储系统/向量加速器**方向直接相关。一般 ECE 同学作为通识选修即可。
