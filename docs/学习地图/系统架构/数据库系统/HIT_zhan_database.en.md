# Harbin Institute of Technology · Zhan Dechen: Database Systems

## Descriptions

- Offered by: Harbin Institute of Technology
- Prerequisites: Data structures, programming
- Programming Languages: SQL
- Instructor: Prof. Zhan Dechen (National First-Class Teaching Achievement Award)
- Difficulty: 🌟🌟🌟
- Estimated Hours: 23 lectures / ~30 hours

Taught by Prof. Zhan Dechen, a National First-Class Teaching Achievement Award winner. Problem-driven, with abundant examples illustrating **relational models, relational algebra/calculus, and SQL** — widely recognized as one of the best Chinese-language database intros.
