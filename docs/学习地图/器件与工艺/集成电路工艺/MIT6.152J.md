# MIT 6.152J：微纳加工技术

## 课程简介

- 所属大学：麻省理工学院（MIT）
- 先修要求：大学物理、半导体物理基础
- 编程语言：无
- 课程难度：🌟🌟🌟🌟
- 预计学时：80 小时

MIT 6.152J / 3.155J《Micro/Nano Processing Technology》是 MIT 电气工程与材料科学双系联合开设的微纳加工技术核心课程，系统讲授现代集成电路制造的全套工艺模块。课程从晶体生长与衬底制备出发，依次深入氧化、扩散、离子注入、化学气相沉积（CVD）、物理气相沉积（PVD）、光刻（光学与电子束）、湿法与干法刻蚀、化学机械平坦化（CMP），直至完整 CMOS 工艺集成流程。每个工艺模块均结合物理机理与工程参数，理论与实践并重。

课程最大特色在于直接与 MIT 微纳加工实验室（MTL）的实际工艺流程结合，讲义中包含大量工业界真实数据与工艺窗口分析。对于希望理解"工艺为何如此设计"而非仅停留在流程记忆的同学，本课提供了严格的物理推导支撑。OCW 开放了完整讲义、习题及部分实验说明，是国内高校"集成电路工艺原理"课程极佳的英文对标参考资源。

该课程在难度与深度上与复旦 MICR130007《集成电路工艺原理》高度契合，建议作为主要英文补充资料。在掌握各工艺模块原理后，可进一步阅读 Plummer 等人的教材《Silicon VLSI Technology》以获得更系统的理论体系，并以本课 OCW 讲义作为精读材料进行对比学习。
