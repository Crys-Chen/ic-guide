# 复旦：集成电路制造仿真模拟原理和应用

## 课程简介

- 所属大学：复旦大学
- 课程代码：ICSE30014
- 学分/学时：2 学分 / 36 学时
- 修读要求：选修（芯粒集成模块）
- 考核方式：期末论文+课堂小测
- 任课教师：俞少峰、朱小娜
- 课程难度：🌟🌟🌟

    本课程为工艺器件方向的专业进阶选修课，主要聚焦于对半导体制造仿真TCAD原理的介绍。课程分为器件仿真和工艺仿真两大部分，详细介绍了各种工艺流程、器件结构及二者的数学建模。工艺和器件背景介绍方面的内容与半导体器件+集成电路工艺原理大多重合，而TCAD原理部分的内容则是未来研究器件设计方向的重要知识积累。课件和课堂小测为全英文，授课讲述为中文。

## 课程资源
 - 课程教材：3D TCAD Simulation for semiconductor processes, devices andoptoelectronics(Li, Simon, Fu, Yue)
