# 嵌入式 SoC

嵌入式 SoC(Embedded System on a Chip)指**把 CPU、内存、外设、加速器集成到单一芯片上,嵌入到特定设备执行专门任务**。智能手机的主处理器、路由器的控制芯片、汽车 ECU、IoT 设备都依赖嵌入式 SoC。

这是连接"芯片设计"与"系统应用"的桥梁:你写 RTL,但需要懂操作系统、外设驱动、实时性、低功耗设计才能设计一个真正能用的 SoC。
