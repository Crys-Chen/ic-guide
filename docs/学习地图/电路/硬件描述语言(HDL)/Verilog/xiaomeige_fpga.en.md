# Xiaomeige: Zero-to-FPGA / Verilog (2024 Edition)

## Descriptions

- Offered by: Training company (Zhixin / Xiaomeige team, popular industry intro)
- Prerequisites: None
- Programming Languages: Verilog
- Instructor: Xiaomeige
- Difficulty: 🌟🌟
- Estimated Hours: 169 episodes / ~80 hours

Bilibili's highest-traffic zero-to-FPGA Verilog course, frequently recommended within the Chinese FPGA community. Covers the full chain: dev boards, toolchain, syntax, and project pipeline — from LED blinking to SDRAM controllers and image-processing projects.

Complements HDLBits / Nandland by supplying the **"hands-on to actually-on-board"** path: HDLBits drills syntax, Xiaomeige drills engineering.
