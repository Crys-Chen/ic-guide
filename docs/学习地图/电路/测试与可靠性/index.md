# 测试与可靠性

芯片造出来要证明它能用、能一直用。测试覆盖从晶圆级 CP 到成品 FT 的量产环节，可靠性研究器件和电路的退化机制。这是数字验证之外另一大就业岗位类别，模拟/射频岗位尤其看重。数字侧的功能验证（SV/UVM）见[数字验证](../数字设计/数字验证/index.md)槽位。
