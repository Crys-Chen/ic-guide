# MIT 6.011: Signals, Systems and Inference

## 课程简介

- 所属大学：麻省理工学院（MIT）
- 先修要求：信号与系统（MIT 6.003 或等效）、概率论基础
- 编程语言：MATLAB / Python（部分习题）
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：80 小时

MIT 6.011 是 6.003《信号与系统》的直接延续，将 LTI 系统分析与概率统计推断深度融合。在傅里叶/拉普拉斯/Z 变换框架的基础上，课程引入随机过程（广义平稳过程、功率谱密度）、维纳滤波与最优线性估计、假设检验与信号检测（Neyman-Pearson 判决准则），最终延伸至信道容量与编码理论初步。

对于微电子方向的学生，本课程是理解 ADC/DAC 噪声分析、无线接收机灵敏度计算以及模数混合信号设计中各类性能折中的理论基础。OCW 提供完整讲义及历年习题集。
