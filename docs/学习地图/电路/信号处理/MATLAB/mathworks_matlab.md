# MathWorks MATLAB Onramp（官方免费课程）

## 课程简介

- 所属大学：MathWorks 官方
- 先修要求：无（高中数学水平即可）
- 编程语言：MATLAB
- 课程难度：🌟🌟
- 预计学时：2–4 小时（Onramp）；按需扩展至 20+ 小时

MathWorks 官方的 MATLAB Onramp 是零基础入门 MATLAB 的最高效路径：无需本地安装，在浏览器内完成约 2 小时交互式练习即可掌握向量/矩阵运算、绘图、函数定义与脚本编写。复旦校园网用户可通过学校 MathWorks 授权账号免费访问完整功能。

完成 Onramp 后，MATLAB Academy 还提供"Signal Processing with MATLAB"等专项课程，可按专业方向按需选取。如需更系统的编程基础，可参考 Coursera Vanderbilt MATLAB 课程（独立页面）。
