# Stanford：Boris Murmann ADC 设计教程

## 课程简介

- 所属大学：斯坦福大学（Stanford University）
- 先修要求：模拟集成电路设计、数字信号处理基础
- 编程语言：MATLAB / Cadence（仿真验证）
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：40–60 小时

Boris Murmann 教授长期维护全球最权威的 ADC 性能数据库（ADC Performance Survey），汇总了 1997 年至今数千篇 ADC 论文的能效指标（Walden FOM / Schreier FOM），并在 ISSCC 等顶会 Tutorial 中配套发布系统性讲义，深度解析 SAR、Pipeline、Sigma-Delta 等主流 ADC 架构的性能极限与工艺趋势。

这些公开 Tutorial 讲义（可通过搜索"Murmann ADC tutorial slides"获取）是目前系统性学习数据转换器设计最高质量的免费英文资源。核心参考教材为 Franco Maloberti《Data Converters》（Springer）及 Razavi《模拟 CMOS 集成电路设计》第 14–15 章。

## 学习地图

- ADC Performance Survey（FOM 数据库，持续更新）：<https://web.stanford.edu/~murmann/adcsurvey.html>
- 课程视频：YouTube 搜索 "Murmann ADC tutorial" 可找到 ISSCC Short Course 录像
- 课程教材：《Data Converters》Franco Maloberti（Springer）
- 补充教材：《Understanding Delta-Sigma Data Converters》Schreier & Temes（IEEE Press）
