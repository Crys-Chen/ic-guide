# 上海交通大学 郑益慧：模拟电子技术基础

## 课程简介

- 所属大学：上海交通大学
- 先修要求：电路原理
- 编程语言：无
- 授课教师：郑益慧
- 课程难度：🌟🌟🌟
- 预计学时：44 讲

B 站口碑顶级的中文模电课，被广泛认为"讲课水平略胜清华华成英版"。郑益慧老师小信号等效模型讲得透，半导体器件物理图像清晰，是中文模电入门的最佳搭档。

配套教材是童诗白/华成英《模拟电子技术基础》（高教社），可与清华华成英版互补观看。
