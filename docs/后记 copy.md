# 后记


> 国内绝大部分大学的本科教学，不是濒临崩溃，而是早已崩溃。————《上海交通大学学生生存手册》

> 本科教育至少得展现它应有的诚意，一种分享知识的诚意，一种以人为本的诚意，一种注重学生体验的诚意。它至少不应该是一种恶意，一种拼比知识的恶意，一种胜者为王的恶意，一种让人学无所得的恶意。但这一切能改变吗？我不知道。————《CS自学指南》

最初北大那一份「CS自学指南」，是我2023年在打复旦系际辩论赛时了解到的。当时我们遇到有一道辩题是“当今中国，加强通识教育/专业教育更有利于大学生未来发展”，我们的持方是反方，也就是“专业教育”。当时我们有一条论证路径是“缺啥补啥”：当今中国内地的本科教育，根本无法给人才提供正常的学科训练和专业培养。连北大这种顶级学府的 A+ 学科都需要“CS自学指南”来自救。

从2024年初开始，我有了仿照那份自学指南自制 IC自学指南的想法。断断续续做了两年，断断续续。最初只是想搭一个个人网站，分享一些高年级的认知和学习笔记，仅局限于复旦校内。慢慢地我觉得，这样没意思。在这个专业里受苦受难的不止复旦的学子。如果一个从社交媒体慕名而来的同学，点开网站，发现这些资源都是复旦专属，那该有多失望！

“我想成为的人”和“系统引导我成为的人”之间存在着很大的张力。有些管教育的，嘴上说要培养什么样的人，实际上他们制订的制度，设计的培养方案，以及相应的激励机制，南辕北辙。单纯从培养方案来看，从系统上来看，就是要我们面面俱到。可“生也有涯，知也无涯”，如果我照着系统的要求，每一门课都付出十足的精力去学，去拿高分，那我的生活怎么办？

搭建网页对我来说很困难，学习前端的知识，UI的设计，要用衬线字体还是非衬线字体，按钮要圆角还是？非常繁琐，做一个轮播图，就过去了一天。直到claude code的出现。

我记得去年暑假，我还在坚持手撸代码。那时的AI会写代码，但写完需要人类去debug，整体效率没有提升。今年已经可以自动搭建一个网页。但还是比较难受的。无论是画图还是做网页，AI都只能快速搭建一个80%的东西，剩下20%，如果完全不懂web开发，不懂代码，就只能用prompt去调，这同样是旷日持久的工程。

“本科教育”是什么？“本科”与“教育”两个词，可能已经不适合再放在一起看。据我所知

“国内绝大部分大学的本科教学，不是濒临崩溃，而是早已崩溃”————这句话诞生于2008年。敢问今夕是何年？情况是否有所好转？诸君冷暖自知。我在国外游历过一些学校，发现其实欧美的本科教育其实也在崩溃。世道如此，倒也不必苛责系统里处于某个位置的人。只是“本科”与“教育”这两个词，或许越来越不适合放在一起思量。现在大学还能提供给我们什么呢？有人说，一个理想的大学就应该是一个自助餐厅，尽可能多的给钱给资源，不要再想着教学生成为什么样的人。“本科教育”？只有学工那群做宣传口的还在琢磨这种抽象的东西。但我觉得“师者，所以传道受业解惑也“。可如今，工科 90% 的线下课程都不如网课，99%的助教都没有 AI 耐心，”受业“与“解惑”已经失守了。可还有“传道”。本科期间我还是遇到过不少老师，他们能够扛起“传道”的大旗。比如谢锡麟老师、郑益薏老师（交大）、易婷老师、蒋玉龙老师、严昌浩老师、叶凡老师、茹国平老师、许灏老师、王伶俐老师。这些老师在上课时是嘴角上扬的。或许不太需要追求效率。就像写字一样，现在对于很多人来说，已经只是一项纯粹的娱乐活动。上课也一样，学知识有很多种方式，上课只是其中一种。上这些老师的课，是一种纯粹的享受。


《上海交通大学学生生存手册》与《CS自学指南》，当初两位前辈创作这两份作品的时候，是带着愤怒与无奈的。与他们不同的是，我倒没有。对于系统本身，我无意改变什么。设计这个网站让我很开心；把在脑海里积攒多年的信息倒出来让我很开心；把每一个科研方向用讲故事的方式说清楚很开心；写下这段文字让我很开心；如果屏幕前的有缘人发现觉得网站里的东西有用，那我也很开心。
