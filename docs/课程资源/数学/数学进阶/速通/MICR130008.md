# 复旦：工程数学及概率方法

## 课程简介

- 所属大学：复旦大学
- 先修要求：数学分析、线性代数
- 编程语言：无
- 任课教师：荆明娥
- 课程难度：🌟🌟🌟

复旦微电子学生在学完数学分析和线性代数后唯一一门也是最后一门数学课，讲得比较简单，也比较枯燥，但很有用。其中涉及到的复变函数和积分变换是信号处理的基础，而概率论则是打开机器学习和深度学习大门的钥匙。

## 推荐公开课

本课程横跨三个模块，本站已为各模块收录专项资源：

- **复变函数与积分变换**：参见 [Bilibili 复变函数与积分变换](../复变函数/bilibili.md)
- **概率论（离散方向）**：参见 [MIT 6.042J](6.042J.md) 与 [UCB CS70](CS70.md)（均含完整概率论章节）

### 英文推荐（概率论）：MIT 6.041 Probabilistic Systems Analysis（OCW）★ 强烈推荐

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/6-041-probabilistic-systems-analysis-and-applied-probability-fall-2010/>
- YouTube 视频：<https://www.youtube.com/playlist?list=PLUl4u3cNGP61MdtwGTqZA0MreSaDybji8>
- 课程难度：🌟🌟🌟🌟
- 简介：MIT 概率论专项课（John Tsitsiklis 主讲，MIT 工程系最受好评课程之一），系统讲授样本空间与事件、条件概率与 Bayes 定理、随机变量（离散/连续分布）、期望与方差、大数定律与中心极限定理，有完整视频。内容直接对应 MICR130008 概率论模块，且为后续机器学习、信号处理奠定最坚实的基础，强烈推荐超前学习。

### 英文推荐（复变函数）：MIT 18.04 Complex Variables with Applications（OCW）

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/18-04-complex-variables-with-applications-spring-2018/>
- 课程难度：🌟🌟🌟🌟
- 简介：MIT 工程应用复变函数课，覆盖复数运算与复平面、解析函数（Cauchy-Riemann 方程）、复积分（Cauchy 定理与 Cauchy 积分公式）、幂级数展开、留数定理与围道积分，内容与 MICR130008 复变函数模块完全对应，是工科复变学习的最优英文参考资料。

### 中文推荐（概率论）：浙大版 概率论与数理统计（中国大学 MOOC）★ 强烈推荐

- 平台：中国大学 MOOC / B 站
- 搜索方式：中国大学 MOOC 搜索 "概率论与数理统计"（浙大版）；B 站搜索 "概率论 浙大"
- 课程难度：🌟🌟🌟
- 简介：以盛骤等《概率论与数理统计》（国内最主流概率教材）为主线，系统讲授随机事件与概率、条件概率与全概率公式、随机变量及其分布、期望与方差、大数定律与中心极限定理，是 MOOC 上最受好评的概率论课程，是 MICR130008 概率模块的首选中文配套资料。
