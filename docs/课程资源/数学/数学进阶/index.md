# 数学进阶

数学进阶覆盖**研究方向需要的专门数学工具**:概率统计、信息论、矩阵分析、凸优化、离散数学、数值分析、密码学等。这些不是所有人都要学,而是**按方向反推**需要哪些数学。

## 子目录

- **[入门速成](入门速成/)** — 工程数学、CS70、6.042J 等"压缩版"通识课
- **[矩阵分析](矩阵分析/)** — MIT 18.065(Strang 新版)、HIT 严质彬;ML/信号处理必备
- **[高等代数](高等代数/)** — 北大丘维声;比线代更严谨,适合理论研究
- **[抽象代数](抽象代数/)** — Harvard;密码学、编码理论的前置
- **[密码学](密码学/)** — Stanford;硬件安全方向直接相关
- **[信息论](信息论/)** — 数据压缩、信道容量;通信、ML、安全的共同基础
- **[离散数学](离散数学/)** — UCB CS126;图论、组合、概率
- **[数值分析](数值分析/)** — MIT 18.330;科学计算的算法基础
- **[凸优化](凸优化/)** — Stanford EE364A(Boyd);ML/EDA/信号处理必备 ★
- **[复变函数](复变函数/)** — B 站资源;射频、信号处理基础

## 学习路径建议

**按科研方向反推**:
- 做 ML/AI:概率论 + 矩阵分析 + 凸优化
- 做模拟/射频:复变函数 + 信号处理基础
- 做 EDA:离散数学 + 凸优化 + 数值分析
- 做硬件安全/密码:抽象代数 + 密码学 + 信息论
- 做量子计算:线性代数(深入)+ 概率 + 量子力学

不必把所有进阶数学都学一遍。
