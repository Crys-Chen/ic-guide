# MIT 18.700: Linear Algebra

## 课程简介

- 所属大学：MIT
- 先修要求：MIT 18.02（多变量微积分）
- 编程语言：无
- 授课教师：MIT 数学系
- 课程难度：🌟🌟🌟🌟
- 预计学时：一个学期 ~36 学时

MIT 数学系本科 proof-based 线代课，定位是 18.06 的"理论加强版"——用 Sheldon Axler 的《Linear Algebra Done Right》为教材，从抽象向量空间公理出发，弱化矩阵机械运算，强调内积空间、谱定理、Jordan 标准型的数学结构。对应中国本科"高等代数（下）"的抽象层次，是想从线代过渡到抽象代数的最佳衔接课。

OCW 提供完整 lecture notes、习题集、答案，但**没有视频**（这是 18.700 的最大遗憾）。建议配合 Axler 自己在 YouTube 上的免费配套视频学习。
