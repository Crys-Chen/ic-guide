# USTC: Numerical Analysis (38 Lectures)

## Descriptions

- Offered by: University of Science and Technology of China
- Prerequisites: Mathematical analysis, linear algebra
- Programming Languages: MATLAB / Python (for experiments)
- Difficulty: 🌟🌟🌟
- Estimated Hours: 38 lectures

USTC's undergraduate numerical analysis course, fully recorded. Systematically covers **error analysis, interpolation, numerical integration, iterative methods for linear systems, nonlinear equation solving, and numerical ODEs**. Paced moderately with clear proofs — a solid Chinese-language foundation for MIT 18.330.

Originally hosted on Youku as a 38-lecture series, mirrored widely on Bilibili.
