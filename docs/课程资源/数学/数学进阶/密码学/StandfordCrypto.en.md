# UCB CS126 : Probability theory

## Descriptions

- Offered by: Standford
- Prerequisites: Abstract Algebra
- Instructor: Dan Boneh
- Programming Languages: None
- Difficulty: 🌟🌟🌟

An essential course for beginners in cryptography. The instructor, Dan Boneh, is from Stanford University. He is an educator and researcher in applied cryptography and computer security, a pioneer in cryptographic pairing, and has trained many leading cryptography experts.
