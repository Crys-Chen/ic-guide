# 数学基础

数学基础是**所有理工科研究的共同起点**:数学分析(微积分)和线性代数。不管你做什么方向,这两门是绕不开的。

## 子目录

- **[数学分析](数学分析/)** — 极限、导数、积分、级数;函数空间的严格分析
- **[线性代数](线性代数/)** — 向量空间、矩阵、特征值、SVD;ML/EDA/信号处理的共同语言

## 学习路径建议

国内的"高等数学 + 线性代数"和"数学分析 + 高等代数"是两套体系,数学分析比高数严格但深;**ECE 学生通学高数+线代即可**,数学分析适合想做理论研究的同学。

强烈推荐 **MIT 18.01/18.02 (微积分) + MIT 18.06 (线性代数,Gilbert Strang)**——Strang 教授的线性代数被誉为"线代讲得最好的人",YouTube + B 站全公开。

## 对应的科研方向

所有[科研方向](../../../科研方向/)的共同前置——任何方向都需要扎实的微积分 + 线代。
