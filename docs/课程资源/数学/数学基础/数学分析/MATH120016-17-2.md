# 复旦：数学分析B

## 课程简介

- 所属大学：复旦大学
- 先修要求：无
- 编程语言：无
- 任课教师：陈纪修
- 课程难度：🌟🌟🌟🌟

陈老师是复旦数学系的泰斗，其编修的《数学分析》可以说是内地最通俗易懂又不失严谨的教材了。与谢老师“从一般到特殊”的教学方式不同，陈老师遵循“从特殊到一般”的顺序，从一些特殊例子出发，一步一步引导我们推导出一般的定理，这更加符合正常人类的学习曲线，学起来也更有成就感。

### 英文推荐（单变量）：MIT 18.01SC Single Variable Calculus（OCW）

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/18-01sc-single-variable-calculus-fall-2010/>
- 课程难度：🌟🌟🌟
- 简介：MIT 单变量微积分，覆盖极限、连续、微分（含隐函数/指数/对数/三角函数求导）、积分技巧（换元/分部/三角代换）与泰勒级数，内容与数学分析B上册核心部分高度对应，讲义清晰简洁，适合与陈老师教材对照参考。

### 英文推荐（多变量）：MIT 18.02SC Multivariable Calculus（OCW）

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/18-02sc-multivariable-calculus-fall-2010/>
- 课程难度：🌟🌟🌟
- 简介：MIT 多变量微积分，系统讲授偏导数、梯度与方向导数、极值与 Lagrange 乘数法、多重积分（含换元/极坐标）、Green 定理、Stokes 定理与 Gauss 散度定理，与数学分析B下册多元微积分部分直接对应。
