# 复旦：数学分析B

## 课程简介

- 所属大学：复旦大学
- 先修要求：无
- 编程语言：无
- 任课教师：谢锡麟
- 课程难度：🌟🌟🌟🌟🌟

谢老师的数学分析在复旦堪称一绝，在国内高校纷纷采用欧美教材的当下，谢老师承袭自苏联的教学体系可谓独树一帜。谢老师对展开有着非常深入的研究，其教授的展开方法我多年以后还在使用，可谓一招鲜吃遍天。该课程难度远超其他校内同名课程，认真学习可以打下坚实的数学分析基础。
