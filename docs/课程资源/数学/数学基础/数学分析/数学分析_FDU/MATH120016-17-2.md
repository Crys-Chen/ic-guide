# 复旦：数学分析B

## 课程简介

- 所属大学：复旦大学
- 先修要求：无
- 编程语言：无
- 任课教师：陈纪修
- 课程难度：🌟🌟🌟🌟

陈老师是复旦数学系的泰斗，其编修的《数学分析》可以说是内地最通俗易懂又不失严谨的教材了。与谢老师“从一般到特殊”的教学方式不同，陈老师遵循“从特殊到一般”的顺序，从一些特殊例子出发，一步一步引导我们推导出一般的定理，这更加符合正常人类的学习曲线，学起来也更有成就感。
