# 复旦：数学分析B

## 课程简介

- 所属大学：复旦大学
- 先修要求：无
- 编程语言：无
- 任课教师：谢锡麟
- 课程难度：🌟🌟🌟🌟🌟

谢老师的数学分析在复旦堪称一绝，在国内高校纷纷采用欧美教材的当下，谢老师承袭自苏联的教学体系可谓独树一帜。谢老师对展开有着非常深入的研究，其教授的展开方法我多年以后还在使用，可谓一招鲜吃遍天。该课程难度远超其他校内同名课程，认真学习可以打下坚实的数学分析基础。

### 英文推荐（单变量）：MIT 18.01SC Single Variable Calculus（OCW）

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/18-01sc-single-variable-calculus-fall-2010/>
- 课程难度：🌟🌟🌟
- 简介：MIT 单变量微积分，系统讲授极限、连续、微分法则（含隐函数、指数/三角函数）、积分技巧（换元/分部/三角）、泰勒级数与 L'Hôpital 法则，内容与数学分析B上册核心内容高度对应。讲义清晰，题目丰富，是工科数学分析课最直接的英文配套资料。

### 英文推荐（多变量）：MIT 18.02SC Multivariable Calculus（OCW）

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/18-02sc-multivariable-calculus-fall-2010/>
- 课程难度：🌟🌟🌟
- 简介：MIT 多变量微积分，覆盖偏导数、方向导数与梯度、二重/三重积分（含坐标变换）、曲线积分与曲面积分、Green/Stokes/Gauss 散度定理，与数学分析B下册（多元函数微积分）直接对应。
