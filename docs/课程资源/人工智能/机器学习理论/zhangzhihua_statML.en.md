# Zhang Zhihua: Statistical Machine Learning

## Descriptions

- Offered by: Shanghai Jiao Tong University / Peking University
- Prerequisites: Advanced probability, linear algebra, intro ML
- Programming Languages: None (theoretical)
- Instructor: Prof. Zhang Zhihua
- Difficulty: 🌟🌟🌟🌟🌟
- Estimated Hours: ~30+ lectures

A classic theoretical course by Prof. Zhang Zhihua. Mathematized machine learning — from **probability inequalities / exponential families / conjugate priors / Wishart distributions** through **RKHS / spectral methods**. The Chinese-language course closest in style to CS229M / 10-708.

Suitable for math / statistics / CS graduate students seeking a deep dive into the mathematical foundations of ML.
