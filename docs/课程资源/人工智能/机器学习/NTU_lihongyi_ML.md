# 台湾大学 李宏毅：机器学习（2023 春季）

## 课程简介

- 所属大学：台湾大学
- 先修要求：线性代数、概率、Python
- 编程语言：Python (PyTorch)
- 授课教师：李宏毅（Hung-yi Lee）
- 课程难度：🌟🌟🌟
- 预计学时：约 64 讲

每年都更新内容，跟得上 AI 圈最新进展。
