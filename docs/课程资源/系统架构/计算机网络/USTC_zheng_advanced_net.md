# 中国科学技术大学 郑烇：高级计算机网络

## 课程简介

- 所属大学：中国科学技术大学
- 先修要求：计算机网络基础
- 编程语言：无
- 授课教师：郑烇
- 课程难度：🌟🌟🌟🌟
- 预计学时：约 50 小时

郑烇老师本科网络课的进阶版，覆盖 **SDN / QUIC / BBR / 多媒体网络 / 移动无线网络**——国内少有的体系化"高级网络"课程。

适合学完哈工大网络课后想深入现代网络的同学。
