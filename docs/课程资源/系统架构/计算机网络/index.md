# 计算机网络

计算机网络研究**计算机之间如何通信**:从物理层(信号传输)到应用层(HTTP/TCP/IP),覆盖以太网、路由、拥塞控制、网络安全等。它是数据中心、5G、IoT 等所有联网系统的共同基础。

对 ECE 学生来说,**做数据中心 / 网络硬件 / 互连 IC** 时直接相关:你设计的 NIC(网卡)、交换机芯片、片上互联(NoC)都要符合网络协议。其他方向作为通识了解即可。
