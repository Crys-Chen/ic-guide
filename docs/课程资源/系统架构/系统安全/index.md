# 计算机系统安全

计算机系统安全研究**软件层的攻击与防御**:漏洞利用、内存安全、网络安全、Web 安全、密码学应用。它从软件视角讲清楚"系统是怎么被攻破的",与硬件安全(侧信道、PUF、HSM)形成互补。

对硬件研究者来说,这门课是**理解侧信道分析、可信硬件设计**的软件视角支撑——你设计的硬件最终运行在 OS 之上,系统漏洞会让硬件安全机制失效。
