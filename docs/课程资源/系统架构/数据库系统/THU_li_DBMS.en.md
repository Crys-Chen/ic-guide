# Tsinghua · Li Guoliang: Database Management Systems

## Descriptions

- Offered by: Tsinghua University
- Prerequisites: Database fundamentals, operating systems
- Programming Languages: C++ (HuaDB labs)
- Instructor: Prof. Li Guoliang (ACM/IEEE Fellow)
- Difficulty: 🌟🌟🌟🌟
- Estimated Hours: ~40 hours

Taught by Prof. Li Guoliang (ACM/IEEE Fellow). Focused on **building** databases — storage engines, transactions, concurrency control, crash recovery, optimizer/executor. Paired with **HuaDB** code and labs, this is essentially the Chinese counterpart to CMU 15-445.
