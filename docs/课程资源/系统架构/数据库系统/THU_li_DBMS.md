# 清华大学 李国良：数据库管理系统

## 课程简介

- 所属大学：清华大学
- 先修要求：数据库基础、操作系统
- 编程语言：C++（HuaDB 实验）
- 授课教师：李国良（ACM/IEEE Fellow）
- 课程难度：🌟🌟🌟🌟
- 预计学时：约 40 学时

李国良教授（ACM/IEEE Fellow）主讲，重在"**造**数据库"——存储引擎、事务、并发控制、故障恢复、优化器/执行器。配套 **HuaDB** 代码与实验，是中文版的 CMU 15-445。
