# 复旦：计算机组成原理

## 课程简介

- 所属大学：复旦大学
- 先修要求：数字逻辑基础、C 语言
- 编程语言：Verilog / C
- 课程难度：🌟🌟🌟🌟
- 预计学时：48 学时

计算机组成原理（MICR130038）面向微电子学院学生，讲授处理器与计算机系统的硬件实现原理：指令集架构（ISA）、数据通路设计、流水线、存储层次（Cache/虚存）、总线与 I/O。学完本课，学生能理解 CPU 如何执行程序，并为进一步学习 SoC 设计、嵌入式系统和 RISC-V 处理器实现打下基础。

- 所属大学：UC Berkeley
- 课程主页：<https://cs61c.org/>（每学期更新）
- 课程难度：🌟🌟🌟🌟
