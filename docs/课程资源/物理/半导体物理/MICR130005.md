# 复旦：半导体物理

## 课程简介

- 所属大学：复旦大学
- 先修要求：量子力学、固体物理
- 编程语言：无
- 授课教师：蒋玉龙
- 课程难度：🌟🌟🌟🌟🌟

蒋玉龙老师是复旦微电器件方向的领军人物之一，相传其在博士期间就已经把评教授需要的论文都发完了。蒋老师对半导体物理的钻研十分精深，其PPT也浓缩了教材的精华，非常适合工科学生学习。

## 推荐公开课

### 英文推荐：MIT 6.012 Microelectronic Devices and Circuits（器件物理部分）

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/6-012-microelectronic-devices-and-circuits-spring-2009/>
- 课程难度：🌟🌟🌟🌟🌟
- 简介：MIT 6.012 前半段讲授半导体物理（载流子统计、pn 结、BJT/MOSFET 物理），视角与蒋玉龙老师课程高度契合，从载流子输运出发严格推导器件特性，讲义质量极高，推荐作为课程的英文深化资料。

### 英文推荐（纳米尺度前沿）：Purdue ECE606 Solid State Devices（Mark Lundstrom）

- 平台：nanoHUB（免费）
- 课程主页：<https://nanohub.org/courses/ECE606>
- 课程难度：🌟🌟🌟🌟🌟
- 简介：Purdue 大学 Mark Lundstrom 教授（固态器件权威）主讲，从量子统计出发推导现代器件工作原理，视角比传统教材更前沿，适合已有一定基础后进一步深化。

### 中文推荐：西安交通大学 半导体物理（B 站）

- 平台：B 站
- 搜索方式：B 站搜索"半导体物理 西安交通大学"或"半导体物理 大学课"
- 课程难度：🌟🌟🌟🌟🌟
- 简介：西安交大等高校的半导体物理课程录像，中文讲解系统完整，与刘恩科《半导体物理学》配套使用，适合与复旦蒋玉龙老师课程互为补充。
