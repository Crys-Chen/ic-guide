# 复旦：量子力学

## 课程简介

- 所属大学：复旦大学
- 先修要求：数学物理方法（复变函数、偏微分方程）、力学
- 编程语言：无
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：64 学时

量子力学是固体物理与半导体器件物理的理论根基。复旦微电子学院开设量子力学课程，核心内容包括波粒二象性、Schrödinger 方程、一维势阱/谐振子/氢原子精确解、算符方法、微扰理论、角动量与自旋，为后续能带理论、隧穿效应（FinFET/TFET）等器件物理奠定基础。
