# MIT 2.051: Introduction to Heat Transfer

## 课程简介

- 所属大学：MIT
- 先修要求：热力学基础(MIT 2.005)、流体力学基础(MIT 2.006)、微积分
- 编程语言：无
- 任课教师：Kripa K. Varanasi
- 课程难度：🌟🌟🌟
- 预计学时：约 18-20 学时(半学期课)

MIT 机械系开设的**本科传热学入门课**,半学期完成,讲清楚"导热(Fourier 定律) / 对流(Nu/Pr/Re 数) / 辐射(黑体定律、视角因子)"三种传热模式,以及如何在工程问题中综合应用。Kripa Varanasi 教授本身做表面物理/液滴动力学研究,讲法贴近工程直觉。

对 ECE 学生来说,这门课的好处是**短而精**——不像机械系那种一学期 50 学时的版本铺得太宽,2.051 在 ~20 学时内把核心物理讲完,非常适合"我做硅光/封装/功率,只想知道热在芯片里怎么走"的学习需求。

- 没视频但讲义+习题足够自学;习题缺可用 Cengel 教材题目补
