# 复旦：固体物理

## 课程简介

- 所属大学：复旦大学
- 先修要求：量子力学、热力学与统计物理
- 编程语言：无
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：64 学时

固体物理是理解半导体材料与器件的理论基石，核心内容包括晶体结构与对称性、声子与晶格振动、自由电子气与能带理论（Bloch 定理、布里渊区）、半导体能带结构、输运现象（漂移/扩散）。对于 IC 方向的学生，固体物理的能带论直接支撑了后续半导体器件的物理图像。
