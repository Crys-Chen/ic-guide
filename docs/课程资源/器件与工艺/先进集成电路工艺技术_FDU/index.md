# 先进工艺技术

先进工艺技术(Advanced Process Technology)指 **7nm 及以下节点的前沿制程**:FinFET → GAA(Gate-All-Around) → CFET、EUV 光刻、High-k Metal Gate、应变硅、SOI、新型互连材料等。这是从"经典 MOSFET 物理"过渡到"现代芯片制造前沿"的进阶课程。

学这门课的同学应该已经掌握[半导体物理](../../物理/半导体物理/index.md) + [半导体器件](../半导体器件/index.md) + [集成电路工艺](../集成电路工艺/index.md) 三门基础。它带你看懂台积电、Samsung、Intel 在 IEDM/VLSI 上发布的前沿论文。
