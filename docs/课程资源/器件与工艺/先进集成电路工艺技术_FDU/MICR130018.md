# 复旦：先进集成电路工艺技术

## 课程简介

- 所属大学：复旦大学
- 先修要求：集成电路工艺原理
- 编程语言：无
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：48 学时

本课程（MICR130018）在集成电路工艺原理基础之上，深入讲授先进 CMOS 节点（28nm 以下）的关键工艺技术：极紫外光刻（EUV）、FinFET / GAA 晶体管结构、高 κ 金属栅（HKMG）、应变硅、多重图形化（SAQP）等。是工艺工程师方向的最高阶课程，也是理解台积电/三星/Intel 先进制程路线图的理论基础。

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/6-152j-micro-nano-processing-technology-fall-2005/>
