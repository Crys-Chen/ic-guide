# 复旦：半导体器件原理

## 课程简介

- 所属大学：复旦大学
- 先修要求：半导体物理、量子力学基础
- 编程语言：无（部分实验可用 SPICE/Python）
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：64 学时

半导体器件原理是复旦微电子学院器件方向的核心专业课，从半导体物理出发推导 pn 结、双极型晶体管（BJT）、MOS 电容与 MOSFET 的工作原理与电流-电压特性，并延伸至短沟道效应、速度饱和等现代 CMOS 器件议题，是模拟电路、数字集成电路设计的直接物理基础。

## 推荐公开课

### 英文推荐：MIT 6.012 Microelectronic Devices and Circuits ★ 强烈推荐

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/6-012-microelectronic-devices-and-circuits-spring-2009/>
- 课程难度：🌟🌟🌟🌟🌟
- 简介：MIT 的器件与电路联合课，前半部分讲授 pn 结/BJT/MOSFET 器件物理（对应本课内容），后半部分讲授小信号模型与放大器电路（对应模拟电路）。讲义极为详尽，习题与考试题开放，是该方向最经典的 MIT OCW 资源。

### 英文推荐（纳米器件前沿）：Purdue ECE606 Solid State Devices（Mark Lundstrom）

- 平台：nanoHUB（免费）
- 课程主页：<https://nanohub.org/courses/ECE606>
- 课程难度：🌟🌟🌟🌟🌟
- 简介：Purdue 大学 Mark Lundstrom 教授（固态器件领域权威）主讲，从量子力学与统计力学基础出发严格推导器件特性，视角比 Neamen 教材更深刻，适合对器件物理有进一步追求的同学。

### 中文推荐：西安交通大学 半导体器件物理（B 站 / 中国大学MOOC）

- 平台：B 站 / 中国大学 MOOC
- 搜索方式：B 站搜索"半导体器件 西安交通大学"或"半导体器件物理 大学"；中国大学MOOC 搜索"半导体器件"
- 课程难度：🌟🌟🌟🌟
- 简介：西安交大等高校的半导体器件课程录像在 B 站和 MOOC 上均有完整版，内容与国内主流教材（刘恩科版）配合，中文讲解易于理解，适合与复旦课程同步参考。
