# 半导体器件：补充资源

## MIT 6.012 配套资源

### NPTEL：IIT Madras 电子器件与电路

- 平台：NPTEL（免费）
- 入口：<https://nptel.ac.in/>（搜索"Electronic Devices and Circuits"）
- 简介：多所 IIT 的电子器件 NPTEL 课程，内容扎实，节奏适中，配合 Sedra/Smith 或 Neamen 教材使用效果良好

### 中国大学 MOOC：国内高校半导体器件课程

- 平台：<https://www.icourse163.org/>
- 推荐院校：东南大学、西安电子科技大学（搜索"半导体器件"或"电子器件"）
- 简介：与国内主流教材（刘恩科《半导体物理学》体系）配套，适合作为中文入门参考
