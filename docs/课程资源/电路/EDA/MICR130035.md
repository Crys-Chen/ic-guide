# 复旦：集成电路计算机辅助设计

## 课程简介

- 所属大学：复旦大学
- 先修要求：数字逻辑基础、模拟电子线路
- 编程语言：Verilog（数字 EDA）
- 课程难度：🌟🌟🌟🌟
- 预计学时：48 学时

本课程（MICR130035）系统讲授集成电路设计自动化（EDA）的核心理论与工具使用，涵盖数字 EDA（逻辑综合、布局布线、时序分析）与模拟 EDA（Spectre 仿真、版图工具），是复旦微电子学生从课程原理走向工程实践的关键一步。

### 英文推荐：VLSI CAD: Logic（UIUC，Coursera）

- 平台：Coursera（免费旁听）
- 课程主页：<https://www.coursera.org/learn/vlsi-cad-logic>
- 课程难度：🌟🌟🌟🌟
- 简介：伊利诺伊大学香槟分校（UIUC）的 VLSI CAD 系列课程第一部分，覆盖布尔逻辑化简、逻辑综合算法（ROBDD、Two-Level/Multi-Level 综合）、技术映射，是理解 EDA 工具内部原理的最佳入门课。配套第二部分"VLSI CAD: Layout"覆盖布局布线。

### 英文推荐（理论深化）：MIT 6.827 Algorithmic Lower Bounds

- 平台：MIT OCW / YouTube
- 备注：EDA 算法基础（NP 完全性、布线复杂度理论）可参考 MIT Erik Demaine 的算法讲座
- 简介：理解 EDA 工具为何需要启发式算法（布局布线是 NP 难问题）需要一定算法理论基础，推荐结合 6.006 算法课补充。

- 平台：B 站
- 搜索方式：B 站搜索"EDA 数字后端"或"Synopsys DC 教程"或"VCS 仿真"
- 课程难度：🌟🌟🌟🌟
- 简介：国内有多位工程师在 B 站分享数字后端工具流（DC 综合 → ICC2 布局布线 → PrimeTime STA）的实战教程，接近工业界实际工作流，是课程理论的重要补充。
