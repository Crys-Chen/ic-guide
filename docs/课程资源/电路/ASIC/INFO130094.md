# 复旦：专用集成电路设计

## 课程简介

- 所属大学：复旦大学
- 先修要求：数字逻辑基础、数字集成电路设计、EDA 工具基础
- 编程语言：SystemVerilog / Verilog
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：48 学时

专用集成电路（ASIC）设计（INFO130094）是复旦微电子学院数字方向最高阶的核心课，涵盖完整的 RTL-to-GDSII 设计流程：RTL 编写与仿真 → 逻辑综合（Synopsys DC）→ 物理实现（布局布线，ICC2/Innovus）→ 静态时序分析（PrimeTime STA）→ 物理验证（DRC/LVS）→ 流片。
