# 复旦：FPGA 数字系统设计

## 课程简介

- 所属大学：复旦大学
- 先修要求：数字逻辑基础、Verilog 基础
- 编程语言：Verilog / VHDL
- 课程难度：🌟🌟🌟🌟
- 预计学时：48 学时

FPGA 数字系统设计（MICR130024）是复旦微电子学院数字方向的核心实践课，在 Verilog 语法基础之上，聚焦 FPGA 设计方法论：时序约束、IP 核集成、AXI 总线互联、片上 SoC 设计，以及在 Xilinx/AMD FPGA 开发板上的调试验证。课程配套 Vivado 工具使用，是数字 IC 方向最重要的动手课程之一。

## 推荐学习路径

1. 先完成 [HDLBits](../数字逻辑基础/HDLBits.md) 练熟 Verilog 语法
2. 参照 Digilent 教程完成"LED 闪烁"等基础 FPGA 项目，熟悉 Vivado 工具流
3. 跟随 EECS151 FPGA Lab 完成更复杂的实验
4. 结合复旦课程要求，在开发板上完成课程 Project
