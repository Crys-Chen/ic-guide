# MIT 6.776: High Speed Communication Circuits

## 课程简介

- 所属大学：麻省理工学院
- 先修要求：模拟电子线路、信号与系统、基础射频概念
- 编程语言：SPICE（仿真实验）
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：120 小时

MIT 6.776 是高速通信电路设计方向的研究生水平课程，由 Hae-Seung Lee 和 Michael Perrott 两位教授联合主讲，定位于无线与宽带数据链路系统中的电路级实现。

课程共 25 讲，内容体系完整：传输线理论与 S 参数 → 宽带放大器设计 → 低噪声放大器（LNA）→ 压控振荡器（VCO）与相位噪声分析 → 混频器（Mixer）→ 整数/分数-N 锁相环（PLL）与频率合成器 → 功率放大器（PA）线性化技术。课程配有 SPICE 仿真 RF 电路实验和 CMOS 功率放大器设计大作业，注重系统级与电路级的协同设计思维。

OCW 版本完整开放全部 25 讲讲义（PDF）、习题集、实验材料及大作业指导，虽无视频录像，但讲义质量极高，是英文互联网上最系统的高速通信电路自学文献之一。
