# 复旦：电路基础

## 课程简介

- 所属大学：复旦大学
- 先修要求：无
- 编程语言：无
- 任课教师：易婷
- 课程难度：🌟🌟🌟🌟🌟
- 预计学时：

易老师是复旦微电子为数不多的全职讲师，常年负责教授《电路基础》《数字逻辑基础》《数字信号处理》《模拟集成电路设计》等多门专业核心课程，是复旦微电人绕不过去的一座大山。其教学水平有口皆碑，美中不足的是易老师的PPT较为简略，需要配合她课上讲解才能明白幻灯片上各元素的含义。

该门电路课程是连接集成电路和高中物理电学的桥梁，也是学习模拟电路的必备基础。通过本门课，大一学生将温习高中物理奥赛的电学部分，并首次接触到相位变换和拉普拉斯变换（Laplace Transformation）————信号处理的必备工具。

## 推荐公开课

### 英文推荐：MIT 6.002 Circuits and Electronics（edX / OCW）★ 强烈推荐

- 所属大学：MIT
- 课程主页（OCW）：<https://ocw.mit.edu/courses/6-002-circuits-and-electronics-spring-2007/>
- edX 系列（可免费旁听）：<https://www.edx.org/xseries/mitx-circuits-and-electronics>
- 课程难度：🌟🌟🌟🌟
- 简介：MIT 电路与电子学核心课，从 KVL/KCL、叠加原理、Thevenin/Norton 等效出发，延伸至储能元件时域响应（RC/RL 电路）、拉普拉斯变换在电路分析中的应用，覆盖复旦电路基础课的全部核心内容。edX 版完整免费，是全球质量最高的电路分析公开课之一。

### 英文推荐：UCB EE16A/B Designing Information Devices and Systems

- 本站已收录：[EE16A&B](../../速通/EE16.md)
- 简介：Berkeley 的电路与信号系统入门课，以矩阵方法系统讲授电路分析（节点电压法、运算放大器），视角更加系统现代，可与复旦电路基础课对照参考。

### 英文推荐（印度顶校）：NPTEL Circuit Theory（IIT）

- 平台：NPTEL / YouTube
- 搜索方式：NPTEL 官网 <https://nptel.ac.in/> 搜索 "Circuit Theory"（推荐 IIT Kharagpur 或 IIT Bombay 版本）
- 课程难度：🌟🌟🌟🌟
- 简介：IIT 系列电路理论课，系统讲授基尔霍夫定律、网络定理（Thevenin/Norton/叠加/互易）、交流稳态分析（相量法与复阻抗）及拉普拉斯变换在电路中的应用，有完整视频，内容与复旦易老师电路基础课高度对应。

### 中文推荐：邱关源《电路》配套视频（中国大学 MOOC / B 站）★ 强烈推荐

- 平台：中国大学 MOOC / B 站
- 搜索方式：中国大学 MOOC 搜索 "电路 邱关源"；B 站搜索 "电路基础 邱关源"
- 课程难度：🌟🌟🌟🌟
- 简介：邱关源《电路》（第五版）是国内使用最广泛的大学电路教材，配套 MOOC 视频系统讲授电路分析（KVL/KCL/叠加原理/Thevenin 定理）、动态电路（RC/RL/RLC 时域响应）及正弦稳态分析（相量法），是非复旦学生学习电路基础的首选中文资料，与易老师课程内容几乎完全对应。
