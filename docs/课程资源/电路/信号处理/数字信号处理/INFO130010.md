# 复旦：数字信号处理

## 课程简介

- 所属大学：复旦大学
- 先修要求：信号与系统
- 编程语言：MATLAB（部分实验）
- 任课教师：易婷
- 课程难度：🌟🌟🌟🌟
- 预计学时：48 学时

数字信号处理（DSP）是信号与系统的离散时间深化课程，核心内容包括 DFT/FFT 算法、数字滤波器（FIR/IIR）设计与实现、多速率信号处理。易老师常年主讲此课，是复旦微电子学生必修专业课之一。标准教材为 Oppenheim & Schafer《Discrete-Time Signal Processing》。
