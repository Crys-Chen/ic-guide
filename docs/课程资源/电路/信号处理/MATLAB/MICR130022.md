# 复旦：MATLAB

## 课程简介

- 所属大学：复旦大学
- 先修要求：无
- 编程语言：MATLAB
- 课程难度：🌟🌟
- 预计学时：16 学时

MATLAB 是工程类专业的标准数值计算与仿真工具，复旦微电子学院在信号处理、电路仿真等多门课程中使用 MATLAB 完成实验。本页汇总最优质的自学资源，帮助同学在课程实验前快速掌握必要操作。

## 推荐公开课

### 英文推荐：MATLAB Onramp（MathWorks 官方）★ 强烈推荐

- 平台：MathWorks 官网（免费，浏览器内在线运行，无需本地安装）
- 课程主页：<https://matlabacademy.mathworks.com/>
- 课程难度：🌟🌟
- 简介：MathWorks 官方出品的两小时互动入门课，覆盖矩阵运算、脚本、函数、绘图等核心操作，全程在浏览器内运行。完成后可继续 Signal Processing Onramp、Control Design Onramp 等专项系列，是最快上手 MATLAB 的路径。

### 英文推荐（系统课）：MIT 6.057 Introduction to MATLAB

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/6-057-introduction-to-matlab-january-iap-2019/>
- 课程难度：🌟🌟
- 简介：MIT 寒假短课（4 讲），覆盖 MATLAB 基本语法、矩阵操作、绘图与调试，附习题与完整代码，适合有编程基础的工科学生快速迁移。

### 中文推荐：B 站 MATLAB 系统教程

- 平台：B 站
- 搜索方式：B 站搜索"MATLAB 教程"，推荐选择播放量 10 万以上、配套代码下载的系列
- 课程难度：🌟🌟
- 简介：B 站有多个高质量 MATLAB 系列，涵盖矩阵运算、Simulink 建模与信号处理工具箱，配合复旦实验课直接上手。

## 推荐学习路径

1. MathWorks MATLAB Onramp（2 小时，建立基础操作感）
2. MIT 6.057 讲义（系统化语法与工程应用）
3. 课程实验中遇到具体工具箱（Signal Processing / Control）时，再参考 MathWorks 各专项 Onramp
