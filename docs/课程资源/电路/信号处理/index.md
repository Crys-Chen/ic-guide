# 信号处理

信号处理研究**信号的数学表示和系统对其响应**——傅里叶变换、拉普拉斯变换、Z 变换、滤波、采样定理。它是通信、雷达、生物医学、音频/图像处理的共同数学基础,也是 ADC/DAC、DSP 等具体模块的理论支撑。

## 子目录

- **[信号与系统](信号与系统/)** — 数学基础:连续/离散信号、卷积、各种变换
- **[数字信号处理 (DSP)](数字信号处理/)** — 用计算机或专用硬件做实际信号处理
- **[ADC / DAC](数模模数转换器/)** — 连接模拟世界与数字世界的桥梁
- **[MATLAB](MATLAB/)** — 信号处理算法的快速原型工具

## 学习路径建议

**信号与系统(基础) → 数字信号处理(应用) + MATLAB(工具) → ADC/DAC(混合信号)** 是常见路径。

学完信号与系统后,直接看 [数字信号处理](数字信号处理/)、[MATLAB](MATLAB/) 任意一个,实战感会快速建立起来。ADC/DAC 是模拟与数字电路的交汇,需要先学[模拟电路](../模拟/)和数字电路。
