### 英文推荐：MIT 6.004 Computation Structures（含完整视频）★ 强烈推荐

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/6-004-computation-structures-spring-2017/>
- YouTube 播放列表：<https://www.youtube.com/playlist?list=PLUl4u3cNGP62WVs95MNq3dQBqY2vGOtQ2>
- 课程难度：🌟🌟🌟🌟🌟
- 简介：MIT 经典系统设计课（Chris Terman 主讲），从 CMOS 晶体管到完整处理器的全栈路线：组合逻辑 → 时序电路 → 有限状态机 → Beta RISC 处理器流水线 → 虚拟内存 → OS 入门，配套在 FPGA 上实现完整处理器的 Project。韩军老师课程在嵌入式处理器设计上与 6.004 内容高度契合，是最直接的英文配套课程。

### 英文推荐：UCB CS61C Great Ideas in Computer Architecture

- 所属大学：UC Berkeley
- 课程主页：<https://cs61c.org/>
- YouTube 历年录像：<https://www.youtube.com/playlist?list=PLhMnuBfGeCDM8pXLpqib90mDFJI-e1lpk>
- 课程难度：🌟🌟🌟🌟
- 简介：Berkeley 的体系结构入门课，以 RISC-V 为主线贯穿 C 语言 → RISC-V 汇编 → 处理器数据通路 → 流水线 → Cache 内存层次 → 并行架构。RISC-V 汇编与处理器设计内容与韩军老师课程直接对应，是学习 RISC-V 处理器体系结构的最佳英文公开课。

### 英文推荐（嵌入式系统实践）：Embedded Systems - Shape the World（UT Austin，edX）

- 平台：edX（免费旁听）
- 课程主页：<https://www.edx.org/learn/embedded-systems/the-university-of-texas-at-austin-embedded-systems-shape-the-world-microcontroller-input-output>
- 课程难度：🌟🌟🌟🌟
- 简介：UT Austin Jonathan Valvano 教授主讲，以 ARM Cortex-M4 微控制器为载体，从汇编语言出发讲授嵌入式 C 编程、GPIO/定时器/UART 外设接口、中断服务程序与实时操作系统（RTOS）基础。对韩军老师课程中 SoC 外设接口与嵌入式软件部分有强烈的工程补充价值。

### 中文推荐：B 站 RISC-V 处理器设计与 SoC 开发

- 平台：B 站
- 搜索方式：B 站搜索 "RISC-V 处理器设计"、"RISC-V 流水线 Verilog" 或 "嵌入式 RTOS FreeRTOS"
- 课程难度：🌟🌟🌟🌟
- 简介：B 站有多位来自高校和工程界的博主分享完整的 RISC-V 五级流水线处理器 Verilog 实现教程，以及 FreeRTOS / RT-Thread 嵌入式操作系统入门视频。内容中文讲解，工程实践感强，是韩军老师课程 Project 开发的最佳中文辅助资料。
