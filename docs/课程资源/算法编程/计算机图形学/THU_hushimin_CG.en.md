# Tsinghua University · Hu Shimin: Foundations of Computer Graphics (National Quality Course)

## Descriptions

- Offered by: Tsinghua University
- Prerequisites: Linear algebra, C++/Python
- Programming Languages: C++ / Python
- Instructors: Hu Shimin (Academician of CAS), Liu Yongjin, Zhang Songhai, Xu Kun
- Difficulty: 🌟🌟🌟
- Estimated Hours: 28 lectures / 32 hours

The national-class quality course led by Prof. Hu Shimin (Academician, CAS), with Tsinghua's top graphics team. The classic alternative to GAMES101 — leaning more toward the mathematical foundations and the traditional rendering pipeline.
