# University of Science and Technology of China · Liu Ligang: Computer Graphics (2020 Undergrad)

## Descriptions

- Offered by: USTC
- Course ID: 00106501
- Prerequisites: Linear algebra, basic differential geometry
- Programming Languages: C++ / Python
- Instructor: Prof. Liu Ligang
- Difficulty: 🌟🌟🌟🌟

The USTC undergraduate course taught by Prof. Liu Ligang (also the instructor of GAMES102), recorded fully during the pandemic. Compared with the GAMES series, this version **leans toward geometry processing and mathematical foundations** — the representative Chinese-language "math-school" graphics course.
