# 复旦：数据结构与算法

## 课程简介

- 所属大学：复旦大学
- 先修要求：程序设计基础（C/C++ 或 Python）
- 编程语言：C/C++
- 课程难度：🌟🌟🌟
- 预计学时：48 学时

数据结构与算法是计算机与电子工程方向的必修基础课。复旦微电子学院开设此课，旨在培养 IC 工程师在 EDA 工具开发、算法优化及嵌入式软件编写中所需的编程能力。核心内容包括线性表、树、图、排序与查找算法及复杂度分析。

## 推荐公开课

### 英文推荐：UCB CS61B Data Structures ★ 强烈推荐

- 所属大学：UC Berkeley
- 课程主页：<https://datastructur.es/>（每学期更新）
- 课程难度：🌟🌟🌟
- 简介：Berkeley 数据结构旗舰课，以 Java 为教学语言，Project 设计精良（含 Git 实现、2D 游戏引擎、搜索引擎等），开放材料最完整，是美国顶尖院校中最优质的数据结构公开课程之一。

### 英文推荐（算法进阶）：MIT 6.006 Introduction to Algorithms

- 所属大学：MIT
- 课程主页：<https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-spring-2020/>
- 课程难度：🌟🌟🌟🌟
- 简介：涵盖动态规划、图算法（BFS/DFS/Dijkstra/Bellman-Ford）、最短路径、最大流等经典算法，附完整讲义与习题答案。与 CS61B 形成良好互补：CS61B 偏数据结构实现，6.006 偏算法理论与分析。

### 中文推荐：邓俊辉《数据结构》（清华大学）★ 强烈推荐

- 平台：中国大学 MOOC / B 站
- 搜索方式：中国大学MOOC 搜索"数据结构 邓俊辉"；B 站搜索"邓俊辉 数据结构"
- 课程难度：🌟🌟🌟🌟
- 简介：清华大学邓俊辉教授主讲，配套自编教材《数据结构（C++ 语言版）》，内容深度与广度均超过大多数同类课程，讲解逻辑严密，是国内数据结构课程的最高水平，强烈推荐。
