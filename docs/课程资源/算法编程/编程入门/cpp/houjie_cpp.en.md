# Hou Jie: Object-Oriented C++ — Advanced Development

## Descriptions

- Offered by: Boolan / Hou Jie
- Prerequisites: Intro C++
- Programming Languages: C++
- Instructor: Hou Jie (侯捷)
- Difficulty: 🌟🌟🌟🌟
- Estimated Hours: Part I 13 lectures + Part II additional

The classic course by Taiwan C++ master Hou Jie. Built around two example classes — `Complex` and `String` — to walk through constructors, copy semantics, virtual functions, and inheritance. The most highly regarded Chinese-language course for "actually understanding C++ classes".

Recommended after a basic C++ course, for learners who want to genuinely internalize object-oriented design.
