# Peking University · Chen Bin: Data Structures and Algorithms (Python)

## Descriptions

- Offered by: Peking University
- Prerequisites: Intro Python
- Programming Languages: Python
- Instructor: Prof. Chen Bin
- Difficulty: 🌟🌟
- Estimated Hours: 16-week classroom recording

A complete classroom recording of Prof. Chen Bin's long-running general-education course at PKU — from computational thinking through Python syntax to data structures, in one continuous arc. The most complete Chinese-language "intro to computing using Python", well suited for non-CS majors who want a systematic introduction.
