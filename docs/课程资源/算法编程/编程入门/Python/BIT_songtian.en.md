# Beijing Institute of Technology · Song Tian: Python Programming (National Quality Course)

## Descriptions

- Offered by: Beijing Institute of Technology
- Prerequisites: None
- Programming Languages: Python
- Instructor: Prof. Song Tian
- Difficulty: 🌟🌟
- Estimated Hours: 11 weeks / ~30 hours

National-class quality course with over 5.5 million cumulative learners. Prof. Song Tian's zero-baseline Python intro built around "understanding and using the computing ecosystem", paired with his Python123 practice platform.

Good Chinese-language stepping stone before CS50P / UCB CS61A for absolute beginners.
