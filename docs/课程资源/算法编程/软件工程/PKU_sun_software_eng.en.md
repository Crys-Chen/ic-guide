# Peking University · Sun Yanchun: Software Engineering

## Descriptions

- Offered by: Peking University
- Prerequisites: Intro programming
- Programming Languages: None
- Instructors: Sun Yanchun, Huang Gang, Liu Xuanzhe
- Difficulty: 🌟🌟🌟
- Estimated Hours: 12 weeks

PKU's first-batch national top-class undergraduate course. Comprehensive introduction to structured / object-oriented / agile methods, and the first domestic course to systematically integrate AI / blockchain / cloud / big data / IoT into modern software engineering.

A good Chinese-industry-perspective complement to MIT 6.031.
