# 一生一芯(OSCC)

**[OSCC: One Student One Chip](https://ysyx.oscc.cc/)** ── 中科院计算所发起的"开放原子开源芯片设计"训练计划,带本科生/研究生**从零完成一颗 RISC-V 处理器的设计与流片**。是国内目前**唯一规模化、可流片**的本科生芯片实战训练项目。

## 这个计划是什么

一生一芯的核心理念非常直接:**让每个有志学芯片的本科生,都能在毕业前流片一颗自己设计的处理器**。

- 由**中科院计算所**包云岗团队发起,自 2019 年起已运行多届
- 历届累计**数千名学生参与,数百颗芯片流片**
- 全程**免费,开放报名**——不依赖学校背景,主要是动手能力筛选
- 完整提供**MPW 流片机会**——通过华夏芯、中芯国际等代工厂的多项目晶圆服务
- 训练完成可直接对接产业,**国内 RISC-V 芯片设计相关公司认可度极高**

## 训练阶段

整个训练分为多个阶段,公开免费的部分是前两阶段:

| 阶段 | 内容 | 时长 |
|---|---|---|
| 预学习 | C 语言、Linux、Git、Verilog 基础 | 2-4 周 |
| 数字电路与计算机系统 | 数字逻辑、计算机组成原理、CSAPP | 1-2 个月 |
| 模拟器 NEMU | C 写一个 RISC-V 模拟器 | 1-2 个月 |
| RTL 实现 NPC | Verilog/Chisel 设计 5 级流水 RISC-V CPU | 2-3 个月 |
| 系统软件 | 跑 RT-Thread / Linux 内核 | 1-2 个月 |
| (高阶)流片 | 时序收敛、版图、DRC/LVS、tape-out | 数月 |

**前 5 阶段完全开放公开学习**:<https://ysyx.oscc.cc/docs/>;通过考核者可进入"高阶"环节获得流片资格。
