# Embodied-AI-Guide(具身智能)

**[Embodied-AI-Guide](https://github.com/tianxingchen/Embodied-AI-Guide)** ── Tianxing Chen 与 Lumina Embodied AI Community 共同维护的**具身智能综合知识库**,GitHub 13.5k stars,2026.1 完成最近一轮大规模重组,**国内中文圈具身智能领域信息密度最高的开源资源**。

- 由**清华、上交、北大、CUHK 等高校研究者主导**,Lumina Embodied AI Community 协作维护

## 推荐路径

**EE 背景同学:**

1. 在本站看 [科研方向 / 具身智能](../科研方向/具身智能.md) 建立"机器人产业链"基本认知
2. 看本站 [人工智能板块](../课程资源/人工智能/index.md) 补 ML/DL 基础(吴恩达 + 李宏毅 + CS231n)
3. 进入 Embodied-AI-Guide 的 Hardware + Infrastructure 板块,看具身智能的硬件全貌
4. 选择具体 IC 切入方向(边缘 AI 芯片 / 机器人 SoC / 传感器 IC),回本站看对应科研方向页深耕

**CS 背景同学:**

1. 直接进入 Embodied-AI-Guide 主入口
2. 学完 Algorithm + Control 后,如果想做端侧硬件协同,再来本站 [AI 系统板块](../课程资源/人工智能/AI系统/index.md) 补硬件视角
